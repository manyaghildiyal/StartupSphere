STARTUPSPHERE

An ML-enabled startup–investor recommendation platform built using the MERN stack and hybrid recommendation system techniques.StartupSphere is a web platform designed to intelligently connect startups with investors using machine learning and recommendation algorithms.

The platform helps:
- Startups gain visibility
- Investors discover relevant startups
- Improve startup-investor matchmaking

🛠️ Machine Learning Concepts Used

- TF-IDF Vectorization
- Cosine Similarity
- Content-Based Filtering
- Collaborative Filtering
- KNN (K-Nearest Neighbors)
- Truncated SVD
- Hybrid Recommendation System
